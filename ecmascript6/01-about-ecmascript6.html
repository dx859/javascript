1.部署 Node.js是服务器端语言，对ES6版本支持更高， 使用nvm（MAC）或nvmw（windows）控制node版本 2.Babel转码器 Babel是一个广泛使用的ES6转码器，可以将ES6代码转为ES5代码，从而在浏览器或其他环境执行。 ---- // 转码前 input.map(item => item + 1); // 转码后 input.map(function (item) { return item + 1; }); ---- 命令行环境安装 $ npm install --global babel-cli $ npm install --save babel-preset-es2015 然后在当前目录下，新建一个配置文件.babelrc。 // .babelrc { "presets": ['es2015'] }
